# Documentation

This directory contains the core documentation for the Harmonia project.

## Files

- **`PRD.md`** - Product Requirements Document. Contains the complete specification of features, user stories, and project goals.

- **`API_SPEC.md`** - API Specification. Documents all API routes, request/response formats, and endpoints for the Next.js API routes.

- **`SCHEMA.md`** - Database/Schema Definition. Defines the data models, database schema, and data structures used throughout the application.

- **`CONTEXT.md`** - High-level Project Context. Provides background information, design decisions, and architectural overview of the project.

- **`STYLE_GUIDE.md`** - Style Guide. Contains UI/UX guidelines, design patterns, and coding style conventions.
