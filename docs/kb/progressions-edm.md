# Common Progressions for EDM and Bass Music

## Natural Minor Progressions

### i–VI–III–VII (A minor)

A classic minor progression used in many EDM and pop tracks:

- **i** = Am = A C E
- **VI** = F = F A C
- **III** = C = C E G
- **VII** = G = G B D

This progression has no dominant tension; all chords are diatonic to A natural minor.

### i–iv–VII–III

Another common minor loop:

- **i** = Am
- **iv** = Dm
- **VII** = G
- **III** = C

### i–VI–iv–V (with harmonic minor V)

When V is major (e.g., E major in A minor), it creates stronger resolution. This uses harmonic minor for the V chord.

## Major Key Progressions

### I–V–vi–IV

The "pop" progression. In C major: C–G–Am–F.

### I–vi–IV–V

Classic doo-wop. In C major: C–Am–F–G.

### ii–V–I

Jazz cadence. In C major: Dm–G–C.

## Roman Numeral Notation

- Uppercase = major chord
- Lowercase = minor chord
- ° = diminished
- 7 = dominant seventh (unless specified maj7 or min7)
